# Interactive Physics Bouncing Balls

A real-time 3D particle playground built for the web. Hover over the balls and watch them scatter, collide, and bounce with realistic physics. The scene is fully interactive, immersive, and optimized for browser performance.

## Features

- **Physics-Driven Motion**: Spheres move, collide, and bounce realistically using rigid body physics.
- **Interactive Hover**: User hover creates chaotic motion, scattering balls in real time.
- **Camera Control**: Movement is limited to horizontal (left-right) panning to keep the focus on the scene.
- **Protected Boundaries**: Hidden walls keep the balls contained within the play area.
- **Real-Time Rendering**: Hundreds of 3D spheres rendered directly in the browser.
